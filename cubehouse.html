Wedding Pavilion on Lake Erie

Design a composition of three mutually perpendicular rectangular planes. Insert these three planes into a 5” cube, using the three planes to create different spaces within the cube. This cube must then be transformed into a 20’ cube wedding pavilion placed on a 80’ x 80’ plot of land facing lake Erie. The site is covered in 8’ tall hedge and carved to create the defined spacial requirements. The main objective of this project is to create a clearly defined hierarchy of space.
